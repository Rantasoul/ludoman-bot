# 🎰 Discord-бот «Лудоман» для Dota 2 лобби

Легковесный, быстрый и производительный Discord-бот, написанный на **Go (Golang)** с использованием библиотеки `discordgo`. Создан специально для автоматизации игрового сервера, сбора лобби 5х5 и верификации участников в уникальном стиле комьюнити.

---

## 🔥 Основные фичи

### 1. Автоматическая верификация игроков
- Бот автоматически отслеживает вход новых участников.
- Отправляет интерактивную кнопку анкеты в приветственный канал.
- При клике открывает модальное окно (3 поля: ник в Steam, реальное имя, MMR).
- Автоматически меняет никнейм на сервере в формате: `Ник | Имя | MMR`.
- Выдаёт роль участника и публикует праздничное приветствие.
- Имеет защиту от слишком длинных никнеймов (лимит Discord — 32 символа).
- **Умная проверка:** если игрок уже зарегистрирован (имеет роль), бот просто обновляет его данные и отправляет уведомление в канал.

### 2. Цветной пульт сбора лобби (`/opros <время>`)
- Команда принимает обязательный аргумент `время` (например: `19:00`).
- Бот тегает роль `Дота5х5` и выкатывает меню с цветными кнопками-эмодзи:
  - ✅ `Я буду` — зелёная
  - 🤡 `Я долбоеб` — красная
  - ⏳ `Позже` — серая
- **Счётчики** обновляются в реальном времени, показывая количество игроков, выбравших каждый вариант.
- **Лок канала:** команду можно использовать строго в канале сбора лобби.
- **Ограничение на переголосование:** игрок может сменить выбор только **один раз** (switch_count < 1).
- **Автоочистка:** через 20 минут после указанного времени бот удаляет:
  - Сообщение с опросом
  - Все фидбек-сообщения (кто куда нажал)
  - Записи из базы данных
  - Отправляет уведомление в канал
- **Личные напоминания:** за 5 минут до сбора бот отправляет ЛС всем, кто нажал "Я буду" с текстом: *"Заходи, тебя все ждут!"*

### 3. Регистрация через команду (`/setup_reg`)
- Любой игрок может вызвать команду и получить кнопку регистрации.
- Сообщение с кнопкой **видит только вызвавший** (эфемерное).
- Удобно для тех, кто пропустил автоматическое приветствие.

### 4. Требования
Установите [Go](https://go.dev) версии **1.25+** на свой компьютер.

### 5. Настройка переменных окружения
Создайте в корне проекта файл .env и заполните:

env
DISCORD_BOT_TOKEN=токен_бота
GUILD_ID=ID_сервера
WELCOME_CHANNEL_ID=ID_канала_приветствий
LOBBY_CHANNEL_ID=ID_канала_лобби
TARGET_ROLE_ID=ID_роли_для_выдачи
DOTA_ROLE_ID=ID_роли_для_тега
DATABASE_URL=postgresql://username:password@host/database?sslmode=require

### 6. Настройка прав в Discord
Включите Developer Mode в настройках Discord.

Пригласите бота на сервер с правами Administrator.

В настройках сервера → Роли перетащите роль бота в самый верх (выше всех ролей, которым он будет менять ники).

Убедитесь, что у роли бота есть права:

Управлять никами

Управлять ролями

Отправлять сообщения

### 7. База данных
Бот использует PostgreSQL. При первом запуске автоматически создаётся таблица lobby_votes для хранения голосов.

### 8. Запуск
bash
go mod tidy
go run .

### 9. Деплой на Render
Залейте код на GitHub.

Создайте на Render Background Worker (не Web Service!).

Укажите:

Build Command: go build -o app

Start Command: ./app

Добавьте все переменные окружения из .env.

Нажмите Deploy.
