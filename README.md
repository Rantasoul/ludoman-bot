# 🎰 Discord-бот «Лудоман» для Dota 2 и Coop

Легковесный, быстрый и производительный Discord-бот, написанный на **Go (Golang)** с использованием библиотеки `discordgo`. Создан специально для автоматизации игрового сервера, сбора лобби 5х5, верификации участников и управления ролями для **Dota 2** и **Coop-игр**.

---

## 🔥 Основные фичи

### 1. Автоматическая верификация игроков
- Бот автоматически отслеживает вход новых участников.
- Отправляет интерактивную кнопку анкеты в приветственный канал.
- При клике открывает выбор игры:
  - 🎮 **Только Dota 2**
  - 🤝 **Только Кооп игры**
  - 🎮🤝 **И Dota 2, и Кооп**
- В зависимости от выбора открывается модальное окно:
  - **Dota / Both:** ник в Steam, реальное имя, MMR → формат `Ник | Имя | MMR`
  - **Coop:** только имя → формат `Имя | Coop`
- Автоматически меняет никнейм на сервере.
- Выдаёт соответствующие роли (`DotaRoleID`, `CoopRoleID`).
- **Умная проверка:** если игрок уже зарегистрирован, бот обновляет его данные и роли в зависимости от нового выбора.
- Разные уведомления в канал при смене игрового статуса.

### 2. Цветной пульт сбора лобби (`/opros <время>`)
- Команда принимает обязательный аргумент `время` (например: `19:00`).
- Бот тегает роль `Дота5х5` и выкатывает меню с цветными кнопками-эмодзи:
  - ✅ `Я буду` — зелёная
  - 🤡 `Я долбоеб` — красная
  - ⏳ `Позже` — серая
- **Счётчики** обновляются в реальном времени.
- **Лок канала:** команду можно использовать строго в канале сбора лобби.
- **Ограничение на переголосование:** игрок может сменить выбор только **один раз**.
- **Автоочистка:** через 20 минут после указанного времени бот удаляет опрос, фидбеки и записи из БД.
- **Личные напоминания:** за 5 минут до сбора бот отправляет ЛС всем, кто нажал "Я буду".

### 3. Регистрация через команду (`/setup_reg`)
- Любой игрок может вызвать команду и получить эфемерную кнопку для регистрации или обновления статуса.

## 🛠 Технологии
- **Язык:** Go (Golang) 1.25+
- **Библиотека API:** [discordgo](https://github.com/bwmarrin/discordgo)
- **База данных:** PostgreSQL (Neon / Render)
- **Менеджер секретов:** [godotenv](https://github.com/joho/godotenv)
- **Хостинг:** Render.com (Background Worker + веб-заглушка)

## Запуск

### 1. Требования
Установите [Go](https://go.dev) версии **1.25+**.

### 2. Клонирование репозитория
git clone https://github.com/Rantasoul/ludoman-bot.git
cd ludoman-bot
3. Настройка переменных окружения
Создайте в корне файл .env и заполните:

env
DISCORD_BOT_TOKEN=токен_бота
GUILD_ID=ID_сервера
WELCOME_CHANNEL_ID=ID_канала_приветствий
LOBBY_CHANNEL_ID=ID_канала_лобби
DOTA_ROLE_ID=ID_роли_Dota_2
COOP_ROLE_ID=ID_роли_Coop
DATABASE_URL=postgresql://username:password@host/database?sslmode=require

4. Настройка прав в Discord
- Включите Developer Mode.
- Пригласите бота с правами Administrator.
- В настройках сервера → Роли перетащите роль бота в самый верх.
- Убедитесь, что у роли бота есть права:
Управлять никами,
Управлять ролями,
Отправлять сообщения

5. База данных
Бот использует PostgreSQL. При первом запуске автоматически создаётся таблица lobby_votes.

6. Запуск
bash
go mod tidy
go run .
7. Деплой на Render
Залейте код на GitHub.

Создайте на Render Background Worker (можно и  Web Service, но придется поддерживать жизнь бота через костыль(пинг)).

Добавьте все переменные окружения из .env.

Нажмите Deploy.
