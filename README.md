# 🎰 Discord-бот «Лудоман» для Dota 2 лобби

Легковесный, быстрый и производительный Discord-бот, написанный на **Go (Golang)** с использованием библиотеки `discordgo`. Создан специально для автоматизации игрового сервера, сбора лобби 5х5 и верификации участников в уникальном стиле комьюнити.

## 🔥 Основные фичи

1. **Автоматическая верификация фриков:**
   * Бот автоматически отслеживает вход новых участников.
   * Отправляет интерактивную кнопку анкеты в приветственный канал.
   * При клике открывает красивое модальное окно (3 поля: ник в Steam, реальное имя, MMR).
   * Автоматически меняет никнейм на сервере в строгий формат: `Ник | Имя | MMR`.
   * Выдает роль участника и публикует праздничное приветствие на весь сервер.
   * Имеет защиту от слишком длинных никнеймов (лимит Discord — 32 символа).

2. **Цветной пульт сбора лобби (`/opros`):**
   * Команда принимает обязательный аргумент `время`.
   * Бот тегает роль `Дота5х5` и выкатывает красивое меню с цветными кнопками-эмодзи (`Я буду`, `Я долбоеб`, `Позже`).
   * **Лок канала:** Команду можно использовать строго в канале сбора лобби. Если прописать её в другом чате, бот выдаст скрытую ошибку, которую видит только отправитель.

## 🛠 Технологии
* **Язык:** Go (Golang) 1.20+
* **Библиотека API:** [discordgo](https://github.com)
* **Менеджер секретов:** [godotenv](https://github.com)

## 🚀 Быстрый запуск

### 1. Требования
Установите [Go](https://go.dev) на свой компьютер.

### 2. Клонирование репозитория
```bash
git clone https://github.com/Rantasoul/ludoman-bot.git
cd ludoman-bot
```

### 3. Настройка переменных окружения
Создайте в корне проекта файл **`.env`** (он автоматически скрыт через `.gitignore`) и заполните своими ID:

```text
DISCORD_BOT_TOKEN=MTA5ODc2... # Токен вашего бота 
GUILD_ID=1234567890...        # ID вашего Discord-сервера
WELCOME_CHANNEL_ID=987654...  # ID канала, где бот будет вешать кнопку регистрации
LOBBY_CHANNEL_ID=554433...    # ID канала, где разрешен сбор лобби 5х5
```

### 4. Настройка прав в Discord
1. Включите **Developer Mode** в настройках Discord.
2. Пригласите бота на сервер с правами **Administrator** и типом интеграции **Guild Install**.
3. В настройках сервера во вкладке **Роли** перетащите роль бота «Лудоман» в самый верх (выше роли верификации и ролей администраторов), иначе Discord заблокирует боту смену никнеймов.
4. Скопируйте ID роли, которую нужно выдавать при регистрации, и вставьте её в `handlers.go` в константу `TargetRoleID`.

### 5. Запуск
Скачайте зависимости и запустите проект:
```bash
go mod tidy
go run .
```
